## Zadání

Zásuvný modul pro QGIS, který umožňuje prohlížení a přidávání tematických map, které jsou běžně dostupné jako samostatné soubory.Buď by mapy byly přístupné offline přímo v modulu nebo by je modul stahoval online.
Uživatel získá přehlednou nabídku map rozdělených do kategorií a může je jedním kliknutím přidat do svého projektu.
Plugin používá JSON soubor, ve kterém jsou definovány jednotlivé vrstvy, jejich typ, URL a popis.

- Cíl č.1: Přehledná nabídka tematických map uspořádaných do kategorií
- Cíl č.2: Načítání vrstev z JSON souboru
- Cíl č.3: Podporování více typů mapových služeb (např. WMS, WMTS, XYZ)
- Cíl č.4: Přidání vybraných vrstev přímo do projektu QGIS
